# 🚀 QUICK START GUIDE

## Open the website in 3 steps:

### Step 1: Extract the ZIP file
Unzip `luxe-interior.zip` to your desired location

### Step 2: Choose a method

**METHOD A — Direct Open (Fastest)**
- Double-click `index.html`
- Opens in your default browser
- ✅ Works immediately

**METHOD B — Local Server (Best)**
```bash
# Navigate to the folder
cd luxe-interior

# Start a server (choose one):

# Python 3
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

Then open: `http://localhost:8000`

### Step 3: Customize

1. **Update Content** — Edit `index.html`
   - Search: `Luxe Interior` → Replace with your business name
   - Search: `+91 98765 43210` → Replace with your phone
   - Search: `hello@luxeinterior.in` → Replace with your email

2. **Change Colors** — Edit `css/style.css` (lines 10-28)
   - `--gold: #c9a84c;` → Your brand color
   - `--obsidian: #0c0b09;` → Background color

3. **Replace Images** — Add your images to `assets/` folder
   - Update image `src` URLs in `index.html`
